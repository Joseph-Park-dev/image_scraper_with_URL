##### 유용한 노트들 #####
1) 구글 검색 URL만 입력해 주세요
핀터레스트등 사이트의 지원은 나중에 업데이트 될것입니다. 언젠가는...

2) 영어 검색 URL만 입력 해주세요
한국어나 다른 언어들에서 작동하긴 하지만 파일을 다운로드 할수 없는등의 
이상한 행동들을 보여줍니다.

3) 법적 책임과 시스템 위험을 고려해서 사용해 주세요
감히 조언드리는데, 현명하십시오
개발자는 이 프로그램의 유저가 야기한 법적 책임 (저작권 등)에 대해서
책임을 지지 않습니다.

감사해요! by JungBae
